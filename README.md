## 📖 Описание проекта

**Movie Library** — это графическое приложение для управления личной коллекцией фильмов. Приложение позволяет добавлять, редактировать, удалять и фильтровать фильмы, а также просматривать статистику по коллекции. Все данные автоматически сохраняются в файл JSON.

---

## ✨ Функциональные возможности

### Основные функции
- Добавление фильмов (название, жанр, год выпуска, рейтинг)
- Редактирование существующих записей
- Удаление отдельных фильмов или всей коллекции
- Автоматическое сохранение данных
- Просмотр статистики

### Фильтрация
- По жанру (выбор из 12 жанров)
- По году выпуска (диапазон лет: от и до)

### Валидация данных
- Название: не пустое, до 200 символов
- Жанр: обязательный выбор из списка
- Год: целое число от 1888 до текущего года
- Рейтинг: число от 0 до 10
